# Keff

Professional expense manager and CRM mobile app for finance advisors and consultants.

## Features
- Expense tracking with categories
- CRM contact management
- Full modules integrated
- CSV import/export
- Offline-first with SQLite
- Cross-platform (iOS/Android) using React Native + Expo

## Tech Stack
- React Native + Expo
- TypeScript
- SQLite (expo-sqlite)
- React Navigation
- React Native Paper
- PapaParse for CSV

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the app:
   ```
   npx expo start
   ```

3. Run on device/emulator using Expo Go app.

## CSV Import/Export
- In Expenses tab: tap "Export CSV" to save expenses to device storage.
- Tap "Import CSV" to import expenses from a CSV file (with headers: amount, category, description, date).
